# mastermind
Finds the best strategies for a specific mastermind clone on the Air Canada in-flight touchscreens.

This mastermind clone is special because it has 6 colors, but for some reason (a bug?) the solution only ever uses the first 4 of those colors.  So the other 2 colors are allowed in guesses, but are never in the solution.

If you limit your guesses to solutions that are consistent with the guesses so far, the best you can do is win in 5 moves.  And you can force a win in 5 moves from literally any starting guess.  (Try `python3 restricted_search.py`.)

But if you allow any guess - and in particular, allow using the colors never-in-a-solution colors - you can force a win in 4!

Here's the full game tree:
```
python3 fixed_search.py
```
